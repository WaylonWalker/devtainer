I am Waylon Walker, a software developer who spends way too much time optimizing my development environment. This site documents my dotfiles, tools, and workflows.

## What I Do

I build software, write about my setups, and share what I learn along the way. Most of my work involves Python, Kubernetes, and cloud infrastructure, but I'm always exploring new tools and techniques.

## This Site

This documentation site is built with [markata](https://markata.dev/) and serves as a living reference for my dotfiles. It includes:

- Tool documentation and setup guides
- Keybinding references for keyboard-driven workflows
- Configuration examples and explanations
