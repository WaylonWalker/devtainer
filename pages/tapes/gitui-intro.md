A demonstration of the complete git workflow using gitui: staging files, writing commits, and pushing changes.

## What is Shown

- Opening gitui in a git repository
- Staging files with `Space`
- Opening commit dialog with `c`
- Writing a commit message
- Submitting with `Ctrl+s`
- Pushing with `p`

## Key Takeaway

gitui provides a fast, keyboard-driven interface for common git operations. Stage, commit, and push without typing git commands or using a mouse.
