A demonstration of Neovim's Language Server Protocol integration for code navigation and go-to-definition.

## What is Shown

- Opening a Python file in Neovim
- Navigating to a function call
- Using gd to jump to definition
- Using Ctrl+o to jump back

## Key Takeaway

LSP transforms Neovim into a full IDE. Jump to definitions, see documentation, and refactor code with built-in intelligence.
