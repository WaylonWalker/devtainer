A demonstration of creating dedicated tmux sessions for individual git repositories, keeping each project's context isolated.

## What is Shown

- Starting in a git repository
- Creating a new session named after the project
- Listing all active sessions
- Opening the editor in the new session context

## Key Takeaway

Dedicated sessions per project keep your workspaces organized. Each git repo gets its own tmux session with the correct working directory.
