A demonstration of using tmux scratch sessions for quick notes, experiments, and temporary work without cluttering your main workspace.

## What is Shown

- Working in a main project session
- Opening scratch session with Alt+G
- Running quick commands in the scratch pad
- Exiting back to the original session

## Key Takeaway

Scratch sessions provide a clean slate for quick experiments. Create one with Alt+G, do your work, then exit back to where you were.
