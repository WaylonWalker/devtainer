A demonstration of using fzf (with atuin) to search command history using Ctrl+R.

## What is Shown

- Opening history search with Ctrl+R
- Typing to filter previous commands
- Navigating results with arrow keys
- Selecting and executing a previous command

## Key Takeaway

History search eliminates retyping. Find any previous command instantly with fuzzy matching.
