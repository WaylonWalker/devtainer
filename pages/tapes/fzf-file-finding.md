A demonstration of using fzf to quickly find and select files with fuzzy matching and Ctrl+T integration.

## What is Shown

- Opening fzf with Ctrl+T
- Typing to fuzzy filter results
- Selecting a file from the filtered list
- The file path is inserted into the command line

## Key Takeaway

fzf's fuzzy finding makes locating files effortless. Type partial names, see results instantly, select with Enter.
