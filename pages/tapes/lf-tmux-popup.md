A demonstration of using lf inside a tmux popup window, showing the seamless integration between file management and terminal multiplexing.

## What is Shown

- Working in a terminal session
- Opening lf (simulating Alt+E popup binding)
- Navigating to a file
- Opening the file
- Returning to the original session

## Key Takeaway

The tmux popup integration allows you to browse files without leaving your current context. Open files, check directories, then return exactly where you were.
