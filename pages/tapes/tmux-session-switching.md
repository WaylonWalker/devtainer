A demonstration of switching between multiple tmux sessions using Alt+Space and session management commands.

## What is Shown

- Multiple sessions running (dotfiles, project1, work)
- Switching between sessions with Alt+Space
- Checking current session context with pwd
- Listing all active sessions

## Key Takeaway

Session switching allows you to work on multiple projects simultaneously. Switch contexts instantly without losing your place in each project.
