A quick demonstration of lf file manager showing basic navigation using vim-style keybindings.

## What is Shown

- Launching lf in the sandbox directory
- Navigating with `j/k` (down/up)
- Entering directories with `l`
- Going back with `h`
- Jumping to bottom with `G` and top with `gg`
- Quitting with `q`

## Key Takeaway

lf provides fast, keyboard-driven file navigation without leaving the terminal. The vim-style bindings make it intuitive for anyone familiar with modal editors.
