A demonstration of using lazydocker to view and manage Docker containers from the terminal.

## What is Shown

- Opening lazydocker
- Navigating the container list
- Viewing container details
- Checking logs

## Key Takeaway

lazydocker provides a complete Docker management interface in the terminal. View containers, images, volumes, and logs without docker commands.
