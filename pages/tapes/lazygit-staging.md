A demonstration of the complete git workflow using lazygit: staging files, writing commits, and pushing changes.

## What is Shown

- Opening lazygit with modified and new files
- Navigating the status panel
- Staging files with Space
- Creating a commit with c
- Pushing changes with p

## Key Takeaway

lazygit's visual interface makes git operations intuitive. Stage with Space, commit with c, push with p—all without typing git commands.
